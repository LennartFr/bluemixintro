  1. IBM Bluemix is a cloud Platform-as-a-Service (PaaS) developed by IBM
  2. Built on open source compute technologies: Cloud Foundry, Docker, and Open Stack
  3. Runs on Softlayer Infrastructure as a Service (IaaS)
  4. Supports Public cloud, Private cloud, On-prem cloud
  5. Supports the common programming languages and cloud services as well as
    integrated DevOps to build, run, deploy and manage applications on the cloud
